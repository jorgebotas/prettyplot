PubliPlots Examples Gallery
===========================

Welcome to the PubliPlots examples gallery! This gallery contains comprehensive
examples demonstrating the various plotting capabilities of PubliPlots.

All examples are generated from Python scripts and can be downloaded and run
directly. Each example is designed to be self-contained and demonstrates a
specific feature or use case.

.. tip::

   All examples can be run directly as Python scripts or in Jupyter notebooks.
   Click on the source code link to download and modify the examples.

Getting Started
---------------

If you're new to PubliPlots, we recommend starting with:

1. :ref:`sphx_glr_auto_examples_plot_bar_plots.py` - Learn the basics of bar plots
2. :ref:`sphx_glr_auto_examples_plot_configuration.py` - Understand configuration and styling
3. :ref:`sphx_glr_auto_examples_plot_scatter_plots.py` - Explore scatter plots and bubble plots

For advanced visualizations, check out:

* :ref:`sphx_glr_auto_examples_plot_venn_diagrams.py` - Set intersection visualization
* :ref:`sphx_glr_auto_examples_plot_upset_plots.py` - Alternative to Venn diagrams for many sets
